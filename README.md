!!!Projeto criado na Imersão Front-End da Alura!!!

🎬 Clone Netflix - Tela de Perfis

Este projeto é uma recriação da tela de seleção de perfis da Netflix, desenvolvida com HTML e CSS. A ideia principal é praticar estruturação de layout, estilização e organização de arquivos front-end.

📸 Preview

Tela com múltiplos perfis simulando a escolha de usuário, semelhante à experiência da Netflix.

🚀 Tecnologias utilizadas
HTML5
CSS3
